
# GISA API Client

Lightweight client for fetching company/legal information from [GISA (Gewerbeinformationssystem Austria)](https://gisa.gv.at).


---

## Installation

Install `gisa-api-client` using pip:

```bash
  pip install gisa-api-client
```



---

## Features

- Fetch Infos about Companies, Legal Persons, Natural Persons, etc.
- Fully typed for easy use
